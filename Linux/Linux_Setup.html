Software Requirements

Open network is recommended while running the Media Server Studio & OpenCV installations as corporte proxy blocks downloading certain files.

  • Install Ubuntu 16.04 by downloading 64-bit ISO from http://old-releases.ubuntu.com/releases/16.04.1/ubuntu-16.04-desktop-amd64.iso
  • Note: While installing, select the option "install Ubuntu alongside with Windows"

    • Set name: intel

    computer name: intel<NUC Number>

    Username: intel

    Password: intel123


  • Install CVSDK, which will internally install Media Server studio also.

    Link: \\10.224.54.1\Raghavendra\VAinstallerSept24_2.tgz

    Note: Use Ctrl+Alt+F1 to switch to one of the classic UNIX terminals and install the above SDK. It won't install properly in GUI based terminal.

    Run the command:

    $sudo python VAinstaller.py --SDK CVSDK

    Note:- If it asks for any overwriting during installation, say 'y' - yes to confirm

    Run the command:

    $./setupvars.sh from VAinstallerSept24 directory

    Reboot the system

  • Download opencv installation script from \\10.224.54.1\Raghavendra\OpenCV to downloads directory
  • Install OpenCV by running the

    $cd '/home/intel<NUC Number>/Downloads/OpenCV/

    $bash install-opencv.sh

  • Copy the workshop zip file or from \\10.224.54.1\Raghavendra\Workshop.zip to /home/Documents/ and unzip it

    • Verification and Validation

    1. Run python

    $idle
    from terminal. Type: import cv2

    2. It should not give any error.

    3. Type: print cv2.__version__

    4. It should display the cv2 version

    5. Execute $vainfo and check if there are no errors

    6. Execute

    $cd /opt/intel/mediasdk/samples/_bin/x64

    $./sample_decode h264 -i ../content/test_stream.264 -hw -vaapi -r

    This should run a video and show the usage of hw and vaapi for decoding frames in terminal.

    5. Shutdown the system, unplug the cables and stick Intel logo provided in the box, on to the NUC and pack the box.
