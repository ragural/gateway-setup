Hardware Requirements

5th Generation Core Processor

1. Unscrew the NUC kit.

2. Upgrade 1 DDR3 RAM in provided slot properly.

3. Insert Solid state drive in the back case.

4. Screw the NUC kit.

5. Power on the NUC with Monitor connected.


Software Requirements

There are manual and automatic named two ways to configure Windows environment for Visual Retail Workshop. Follow any one method.


Automatic Windows Configuration

1. Automatic method involves the windows image created by Macrium Software and placed in \\10.224.54.1\Raghavendra\Visual_Retail_Windows_Image

Copy the above images in suitable storage device

2. Prepare a Rescue disk (USB) using the ISO provided in \\10.224.54.1\Raghavendra\Rescue.iso according to the instructions given in Creating rescue media

Note:- Bootable image can be created easily in Windows using the Rufus software

3. When the system is switched on, press F10 and select the rescue disk to boot from

4. Boot through this rescue disk and restore the above said windows image using the instructions given in Restoring a system image

Note:- Select *-00-06 image as the restore image. In destination local disk, delete all existing partitions.

5. Click Next > Finish > Continue (if any warning appears). Select Close option after verall progress reaches 100%

6. From the Bottom left, select shutdown WinsPE > Select Reboot > Ok.

7. Check if Windows is loading properly. If there is any problem in loading, go back to rescue disk and fix the Windows boot problems.

8. Once Windows loads properly, change the computer name from System setting and username from user accounts in control panel.

Right Click Windows Button > Select System > Change Setting > Change > Computer Name: intel<NUC Number>

Goto Control Panel > User Accounts > Change your account name: intel<NUC Number>

9. Through elevated command prompt run Desktop > Retail > 01-MARS > filterDlls > regdll.bat, execute to register the dlls required for Mars

10. Shutdown the system, unplug the cables and stick Intel logo provided in the box, on to the NUC and pack the box.

Note:- Windows Boot Troubleshoot


Note:-If the automatic configuration fails, you can try manual installation as follows

Manual Windows Configuration

1. Power on the Intel NUC

2. Press F10 and boot from Windows 10 USB prepared using \\10.224.54.1\Raghavendra\en_windows_10_multiple_editions_version_1607_updated_jul_2016_x64_dvd_9058187.iso

3. Install Windows 10.

4. Set Username: intel and Password: intel123

5. Download Workshop contents from \\10.224.54.1\Raghavendra\Bangalore-Workshop

6. Open the command prompt in admin mode and navigate to downloaded Workshop folder

7. Goto 00-Installers, RunMe1.bat to install drivers. Don't restart in between.

8. Reboot the system after RunMe1.bat finishes.

9. Execute RunMe2.bat to install WindowsSDK, Directx 11 and \\10.224.54.1\Raghavendra\en_visual_studio_2010_professional_x86_dvd_509727.iso

Note:- Directx installation may ask for .NET framework 3.5 for which internet connection is a must.

10. Execute RunMe3.bat to unmount the VS image, register Mars dlls, install opencv and Mesh commander.

11. Copy folders 01 - 04 to Desktop > Retail

12. Through elevated command prompt run Desktop > Retail > 01-MARS > filterDlls > regdll.bat, execute to register the dlls required for Mars


Verification and Validation

1. Goto Desktop Retail > 01-Mars > Bin and run MARS.exe and check if one video is running properly.

2. From the windows search panel Run Python's "idle". Type: import cv2

3. It should not give any error.

4. Type: print cv2.__version__

5. It should display the cv2 version

7. Shutdown the system, unplug the cables and stick Intel logo provided in the box, on to the NUC and pack the box.
